# zekaix.app

Fast-paced top-down PvP shooter built for the [CubePath Hackathon 2026](https://github.com/midudev/hackaton-cubepath-2026), organized by CubePath and [midudev](https://twitch.tv/midudev).

![zekaix.app in-game](https://github.com/user-attachments/assets/2a80a402-a547-4670-898e-b9bd35db108c)

## Overview

**zekaix.app** is a browser-based multiplayer shooter focused on quick matches, custom loadouts, instant respawns, and clean arcade-style combat. Players can jump into a live arena, pick a kit, collect pickups, and fight in real time through a minimal HUD designed to feel like a proper game screen.

This project was created as an entry for the CubePath + midudev hackathon.

## Highlights

- Fast-paced top-down PvP combat
- Custom loadouts with weapon, active, and passive selection
- Instant respawn loop for nonstop action
- Pickups and live feed notifications during the match
- Minimal in-game HUD with minimap
- Real-time multiplayer gameplay
